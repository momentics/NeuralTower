# Обложка

3D-модель (рендер) всего узла NeuralTower в прозрачном корпусе. Должно быть видно вертикальные мезонины, водоблоки с трубками и кабели SlimSAS, уходящие вниз.
- 01_neural_tower_render.png

## Сравнение объемов памяти

Инфографика. Слева - одна RTX 5090 (24-32 ГБ), справа - стек NeuralTower (128 ГБ HBM2). Можно добавить полоски пропускной способности памяти (ГБ/с).
- 02_memory_comparison.png

## Схема V-Core (Аэродинамика)

Вид корпуса в разрезе. Синие стрелки заходят с боков в подвал, красные стрелки "выстреливают" вертикально через сопла палубы вверх.
- 03_vcore_airflow_scheme.png

## Топология PCIe и NVLink

Блок-схема. CPU -> Чипы PLX -> 4x V100. И мостики NVLink между картами внутри мезонинов.

# Руководство по сборке несущего каркаса NeuralTower

Сборка рамы из алюминиевого профиля 2020 требует высокой точности. Поскольку система NeuralTower использует принцип разделения зон давления, геометрия каркаса должна быть идеально прямоугольной для обеспечения герметичности прилегания разделительной перегородки (палубы).

## 1. Подготовка компонентов и инструмента

Перед началом сборки необходимо убедиться в чистоте реза всех профилей. Торцы должны быть строго перпендикулярны оси профиля.

**Необходимый инструмент:**
*   Метчик М6 с держателем.
*   Смазочно-охлаждающая жидкость (машинное масло) для нарезки резьбы.
*   Набор шестигранных ключей.
*   Слесарный угольник 90°.

## 2. Нарезка резьбы в торцах профиля

Центральный канал профиля 2020 имеет диаметр, оптимальный для нарезки резьбы М6. Этот метод соединения является наиболее жестким и долговечным.

1.  Резьба нарезается во всех четырех торцах вертикальных стоек (550 мм).
2.  Глубина резьбы должна составлять не менее 25 мм.
3.  При нарезке необходимо делать пол-оборота назад после каждого полного оборота вперед для вывода стружки, используя масло для предотвращения закусывания метчика.

## 3. Этапы сборки и юстировка геометрии

Сборка производится последовательно, начиная с нижнего основания.

### Шаг 1: Нижний силовой пояс
Соберите нижний прямоугольник (основание), соединив профили ширины (350 мм) и глубины (400 мм). На данном этапе винты затягиваются не до конца. В торцы вертикальных стоек вкручиваются виброгасящие ножки.

### Шаг 2: Установка вертикальных стоек и среднего пояса
Установите вертикальные стойки (550 мм) и закрепите средний пояс (опору палубы) на высоте ровно 200 мм от нижнего края стоек. Высота должна быть идентична для всех четырех углов, так как от этого зависит герметичность будущего подвала.

### Шаг 3: Проверка геометрии
Используйте слесарный угольник для проверки каждого угла. Ключевым тестом является измерение диагоналей каждого прямоугольника рамы - разница между диагоналями не должна превышать 1 мм. Только после подтверждения геометрии производится финальная затяжка всех торцевых винтов М6.

### Шаг 4: Внутренние стапели и опоры
Установите внутренние вертикальные стапели (300 мм) для мезонинов. Они крепятся между средним и верхним поясом с помощью литых уголков. Расстояние между стапелями должно соответствовать монтажным отверстиям мезонинов (пока не пришло оборудование и указать точный размер затруднено, так как поставщик не имеет даташитов).

## 4. Подготовка к монтажу палубы

После сборки рамы необходимо проверить плоскость среднего пояса. Приложите алюминиевую перегородку (палубу) к профилю. Между листом и металлом не должно быть видимых зазоров. Все пазы профиля в зоне подвала, которые не будут закрыты панелями, впоследствии будут уплотнены для исключения утечек воздуха.

Правильно собранный каркас NeuralTower способен выдержать статическую нагрузку более 50 кг без деформации, что гарантирует сохранность прецизионных SlimSAS-соединений и герметичность системы охлаждения.

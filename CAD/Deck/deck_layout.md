# Спецификация разделительной перегородки (Палубы)

Разделительная перегородка является ключевым элементом системы управления воздушными потоками V-Core. Она выполняет роль герметичной мембраны между камерой высокого давления (подвалом) и вычислительным отсеком, обеспечивая направленный теплосъем через систему сопел.

## Общие параметры заготовки

*   **Материал:** Алюминиевый лист (сплав АМг3 или аналоги).
*   **Толщина:** 3 мм.
*   **Габаритные размеры:** 350 x 380 мм.
*   **Метод обработки:** Лазерная или гидроабразивная резка.

## Конфигурация прорезей для воздушных потоков

Для охлаждения компонентов, не имеющих водоблоков, в перегородке предусмотрены щелевые сопла (воздушные ножи). Расположение прорезей рассчитано исходя из координат теплонагруженных узлов материнской платы ASUS X99-E WS и мезонинов V100.

### Центральная зона охлаждения процессора
*   **Окно обратной стороны сокета:** 110 x 110 мм. Располагается непосредственно под процессорным разъемом для охлаждения текстолита.
*   **Сопла VRM материнской платы:** Две перпендикулярные щели шириной 12 мм, расположенные вдоль Г-образной цепи питания процессора. Длина щелей составляет 140 мм и 110 мм соответственно.

### Зоны охлаждения мезонинов
*   **Сопла GPU:** Две прорези размером 160 x 10 мм. Располагаются симметрично по бокам от материнской платы непосредственно под платами мезонинов. Дистанция от края сопла до поверхности платы видеокарты должна составлять не более 20 мм для сохранения скорости потока.

## Технологические отверстия для коммуникаций

### Порты данных SlimSAS
*   Для прохода восьми шлейфов SFF-8654 8i предусмотрены четыре окна размером 32 x 15 мм (по два окна на каждый мезонин). 
*   Края окон должны быть скруглены (радиус 2 мм) для предотвращения повреждения оплетки кабелей.
*   После монтажа кабелей окна герметизируются эластичными вставками для предотвращения падения статического давления в подвале.

### Гидравлические магистрали
*   Для прохода шлангов СЖО предусмотрены четыре круглых отверстия диаметром 20 мм под установку проходных фитингов (Fillport) с резьбой G1/4. Это обеспечивает жесткую фиксацию контура и полную герметичность перегородки.

## Инструкция по монтажу

Перегородка устанавливается на средний пояс рамы из профиля 2020. По всему периметру прилегания к профилю наносится самоклеящийся резиновый уплотнитель. Фиксация осуществляется винтами М5 через предварительно просверленные отверстия по краям листа (шаг 100 мм) в Т-образные гайки, установленные в паз профиля.

Все поверхности после лазерной резки должны быть очищены от заусенцев. Рекомендуется окраска порошковой эмалью для обеспечения диэлектрических свойств поверхности.

# NeuralTower

<img src="./Docs/Images/logo.png" width="100" align="left" alt="Логотип проекта NeuralTower"> Настольная рабочая станция на четырёх Tesla V100 SXM2 с общим объёмом видеопамяти 128 ГБ. Собирается для локального инференса и дообучения моделей вроде Llama3 70B без облаков.
<br clear="left" />

## Как охлаждается

1.4 кВт в квартире — это не шутки. Серверные радиаторы V100 рассчитаны на продув турбинами, а нам нужно тихо. Корпус разрезан алюминиевой палубой на два отсека. Снизу вентиляторы гонят воздух через медные радиаторы СЖО, создавая давление. Воздух выходит через сопла в палубе со скоростью 15+ м/с и бьёт точно в VRM материнской платы и мезонинов. Сами GPU и CPU охлаждаются водой.

## Связь между GPU

Внутри каждого мезонина карты связаны NVLink 2.0 (300 ГБ/с). Между мезонинами данные идут по PCIe через мосты PLX с включённым GPUDirect P2P, чтобы обходить системную память.

Система на Gentoo Linux, всё собрано под Broadwell-EP с CUDA 13.x. vLLM настроен с вытеснением кэша на NVMe, так что контекст может превышать 128 ГБ видеопамяти.

## Быстрый старт мониторинга

```bash
cd Software/Linux/scripts
docker-compose up -d
./monitor.sh
```

Отслеживает температуры HBM2 и потребление каждого GPU в реальном времени.

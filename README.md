# NeuralTower

NeuralTower представляет собой открытый проект настольной вычислительной станции, спроектированной для локального обучения нейронных сетей и работы с массивными языковыми моделями. Основная цель разработки - перенести производительность серверного узла в формат персонального рабочего инструмента, способного эффективно и бесшумно утилизировать более киловатта тепловой энергии в условиях офисного или жилого помещения.

## Архитектура и компоновка системы

В основе физического устройства лежит концепция вертикального монолита. Несущий каркас собран из алюминиевого конструкционного профиля сечением 20х20 мм, что обеспечивает необходимую жесткость для удержания тяжелых медных радиаторов и четырех профессиональных ускорителей Tesla V100. Для достижения стабильности при суммарной нагрузке в 1.4 кВт внутреннее пространство разделено на два функциональных яруса герметичной алюминиевой перегородкой.

### Управление воздушными потоками в нижнем отсеке

Нижний ярус функционирует как камера избыточного давления. В нем горизонтально расположены два медных радиатора типоразмера 360 мм с толщиной 60 мм. Шесть высоконапорных вентиляторов постоянно нагнетают воздух внутрь этого объема, создавая так называемый «воздушный подпор». 

Интеграция блоков питания Corsair HX1000 выполнена по изолированной схеме: они размещены так, что забор воздуха происходит из внутренней охлажденной зоны, но их задние панели герметично прилегают к вырезам в корпусе. Это позволяет выбрасывать отработанное тепло от БП напрямую во внешнюю среду, не подогревая вычислительные компоненты и сохраняя высокое статическое давление в камере.

### Технология направленного охлаждения вычислительного ядра

Вычислительное ядро находится в верхнем ярусе, где горизонтально установлена материнская плата ASUS X99-E WS и вертикально закреплены два мезонина с GPU. Проблема перегрева элементов, не покрытых водоблоками (в частности, цепей питания VRM), решена внедрением скоростных «воздушных ножей». 

В разделительной перегородке под каждым мезонином и вдоль радиаторов питания материнской платы прорезаны прецизионные щелевые сопла. Разница давлений между ярусами заставляет воздух выходить из этих щелей со скоростью свыше 20 метров в секунду. Такой направленный поток эффективно разрушает пограничный слой воздуха у радиаторов, обеспечивая интенсивный теплосъем без использования дополнительных шумных вентиляторов в верхней зоне.

### Интерфейсы данных и кабельный менеджмент

Связь между компонентами реализована через восемь прямых линков SlimSAS SFF-8654 8i, обеспечивающих полноскоростной обмен данными по шине PCIe 3.0. Чтобы не препятствовать движению воздуха и избежать нагромождения проводов, шлейфы данных уложены через технологические окна в перегородке и делают плавную петлю в нижнем отсеке. Такое решение позволяет компенсировать избыточную длину кабелей и оставить пространство вокруг ускорителей открытым для вертикальной продувки.

## Технические возможности и потенциал

Использование 22-ядерного процессора Xeon и 128 ГБ оперативной памяти в сочетании с поддержкой NVLink внутри мезонинов превращает NeuralTower в полноценный узел для Deep Learning задач. Общая видеопамять в 128 ГБ HBM2 позволяет работать с современными архитектурами трансформеров, которые не помещаются в память потребительских видеокарт. Единая «земля» системы, обеспеченная через алюминиевую раму каркаса, минимизирует наводки на высокочастотных линиях передачи данных.

## Состав документации в репозитории

В разделах проекта опубликованы чертежи для лазерной резки палубы, спецификации нарезки профиля и 3D-модели внутренних стапелей. Особое внимание уделено электрической части: схемам коммутации двух блоков питания и распиновке переходников с PCIe на CPU 8-pin, необходимых для корректной работы логики мезонинов. Также представлены рекомендации по настройке BIOS для инициализации расширенного адресного пространства видеопамяти.

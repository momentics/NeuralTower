# Bidirectional Bandwidth с учетом GPUDirect P2P и двух чипов PLX

Необходимо разделять пропускную способность на два независимых контура: внутри мезонинов (через NVLink) и между мезонинами (через шину PCIe и чипы PLX).
Поскольку интерфейсы являются полнодуплексными, они могут одновременно передавать данные в обоих направлениях.

## 1. Межмезонинный обмен и связь с CPU (Через шину PCIe 3.0 x16)
Каждая из четырех карт V100 SXM2 подключена через адаптеры SlimSAS в режиме PCIe 3.0 x16.

* В одну сторону (Unidirectional): Теоретическая скорость PCIe 3.0 x16 составляет 15.75 ГБ/с. С учетом накладных расходов протокола (кодирование 128b/130b) реальная чистая скорость передачи данных составляет ~14.2 ГБ/с на одну карту.
* Двунаправленная (Bidirectional) на одну карту: Поскольку PCIe работает в режиме полного дуплекса, карта может одновременно отправлять 14.2 ГБ/с и принимать 14.2 ГБ/с. Суммарный линк для одного адаптера составляет ~28.4 ГБ/с.
* Суммарный предел межмезонинного обмена (Ограничение шины процессора):
Так как плата ASUS X99-E WS подключена к процессору Haswell-E/Broadwell-E по двум аппаратным линкам x16 (по одному на каждый чип PLX), максимальная общая скорость, с которой Мезонин 1 может обмениваться данными с Мезонином 2 (или с системной RAM/NVMe-диском), жестко ограничена пропускной способностью шины процессора:
* В одну сторону: ~28.4 ГБ/с (14.2 ГБ/с через PLX1 + 14.2 ГБ/с через PLX2).
   * Двунаправленная: ~56.8 ГБ/с суммарно на всю систему.

------------------------------
## 2. Внутримезонинный обмен (Через мосты NVLink 2.0)
Внутри каждого мезонина пары карт (GPU 0 $\leftrightarrow$ GPU 1 и GPU 2 $\leftrightarrow$ GPU 3) соединены напрямую через интерфейс NVIDIA NVLink второго поколения. Ускорители V100 в форм-факторе SXM2 имеют по 6 каналов (links) NVLink на каждом чипе.
В стандартных двухчиповых мезонинах V100 SXM2 (Grid/пассивные подложки) карты обычно соединены всеми шестью доступными линками друг с другом.

* На один канал NVLink 2.0: выдает 25 ГБ/с в одну сторону (50 ГБ/с двунаправленной скорости).
* Суммарно на пару GPU (6 каналов):
* В одну сторону (Unidirectional): $6 \times 25 \text{ ГБ/с} = \mathbf{150 \text{ ГБ/с}}$.
   * Двунаправленная (Bidirectional): $6 \times 50 \text{ ГБ/с} = \mathbf{300 \text{ ГБ/с}}$.

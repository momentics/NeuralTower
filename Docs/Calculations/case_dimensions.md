# Расчет габаритов корпуса NeuralTower

Корпус считается от устанавливаемых компонентов, типа соединителей профиля и минимальных монтажных зазоров.

В расчетах используется внутренняя система координат корпуса: `X = 0-500 мм` слева направо, `Y = 0-300 мм` от передней панели к задней, `Z = 0-590 мм` снизу вверх. Нулевая точка находится в нижнем переднем левом внутреннем углу рамы. Палубный пояс установлен на `Z = 260 мм`.

Разделительная палуба выполняется как вкладной лист `458 x 258 x 3 мм`. Она не накрывает профиль сверху, а утапливается внутрь палубного пояса и в координатах корпуса занимает область `X = 21-479 мм`, `Y = 21-279 мм`. Для отверстий палубы используется локальная сетка листа `Xd = 0-458 мм`, `Yd = 0-258 мм`, где `Xd = X - 21`, `Yd = Y - 21`.

## 2. Ширина

Ширина задается не материнской платой, а сочетанием материнской платы и двух боковых мезонинов.

```text
W_motherboard = 305 мм
W_side_bay = 80 мм
W_clearance = 15 мм на сторону

W_internal = 305 + 2 x 80 + 2 x 15 = 495 мм
```

Боковая зона 80 мм на каждый мезонин включает:

- 20 мм максимального расстояния от сопла до охлаждаемой зоны;
- 30-35 мм на плату, водоблок и крепеж;
- 15-20 мм на фитинги, кабели питания и монтажный доступ.

Расчетный внутренний размер округляется до 500 мм. При профиле 20 x 20 мм внешний размер по ширине получается 540 мм.

## 3. Глубина

Глубина в компактной компоновке задается не длиной радиатора, а глубиной материнской платы и монтажными зазорами. Два радиатора Bykski CR-RD360RC-TK60-V2 разворачиваются длинной стороной вдоль ширины корпуса и устанавливаются параллельно в нижнем отсеке. Для радиаторной группы требуется:

```text
W_radiator_group = 395.5 + 30 = 425.5 мм
D_radiator_group = 2 x 122 + 20 + 30 = 294 мм
```

Ширина радиаторной группы меньше расчетной внутренней ширины 500 мм, поэтому радиаторы не увеличивают ширину корпуса. Глубина радиаторной группы также укладывается в компактный габарит 300 мм.

Верхний отсек по глубине проверяется от материнской платы ASUS X99-E WS:

```text
D_motherboard = 267 мм
D_clearance = 30 мм

D_internal = 267 + 30 = 297 мм
```

Расчетный внутренний размер округляется до 300 мм. При профиле 20 x 20 мм внешний размер по глубине получается 340 мм.

Для диагональной установки радиаторов запаса нет: при повороте пары радиаторов на 10-20 градусов глубина footprint вырастает примерно до 360-420 мм, а при 45 градусах - почти до 500 мм. Поэтому единственная компактная рабочая ориентация - строго длинной стороной вдоль ширины корпуса.

Два радиатора размещаются в нижнем отсеке параллельно, длинной стороной вдоль ширины корпуса. По глубине им требуется:

```text
D_radiators = 2 x 122 + 20 + 2 x 15 = 294 мм
```

Это меньше расчетной внутренней глубины 300 мм, поэтому радиаторы укладываются в компактный корпус без диагонального размещения.

## 4. Высота нижнего отсека

Нижний отсек должен вместить радиаторы, вентиляторы, блоки питания и помпы.

```text
H_radiator_stack = 60 + 25 = 85 мм
H_psu_on_edge = 150 мм
H_pump_60_tank = 60 мм резервуар + корпус помпы
H_clearance = 10-15 мм
```

Резервуар Bykski B-UL-D5-X-V4 в этой сборке имеет высоту 60 мм без учета самой помпы. Поэтому высоту нижнего отсека задает не помпа, а блок питания HX1000 на ребре, установленный выше радиаторно-вентиляторного слоя:

```text
H_lower_min = H_radiator_stack + H_psu_on_edge + H_clearance
H_lower_min = 85 + 150 + 15 = 250 мм
```

Для текущей компоновки принимается:

```text
H_lower = 260 мм
```

Радиаторы занимают нижний слой `Z = 0-85 мм`. Блоки питания начинаются только с отметки `Z = 95 мм`, то есть между радиаторно-вентиляторным слоем и PSU-отсеками остается расчетный монтажный зазор `10 мм`. Верх БП находится на `Z = 245 мм`, что дает `15 мм` до палубы.

Расчетные координаты нижнего отсека:

| Узел | Координаты |
| --- | --- |
| Передний радиатор с вентиляторами | `X = 52-448 мм`, `Y = 18-140 мм`, `Z = 0-85 мм` |
| Задний радиатор с вентиляторами | `X = 52-448 мм`, `Y = 160-282 мм`, `Z = 0-85 мм` |
| Левый HX1000 | `X = 0-86 мм`, `Y = 120-300 мм`, `Z = 95-245 мм` |
| Правый HX1000 | `X = 414-500 мм`, `Y = 120-300 мм`, `Z = 95-245 мм` |
| Левая помпа | `X = 160-240 мм`, `Y = 20-95 мм`, `Z = 95-215 мм` |
| Правая помпа | `X = 260-340 мм`, `Y = 20-95 мм`, `Z = 95-215 мм` |
| Подпалубный кабельный коридор | `X = 90-410 мм`, `Y = 21-115 мм`, `Z = 225-235 мм`; на листе палубы `Xd = 69-389 мм`, `Yd = 0-94 мм` |

PSU-отсеки являются изолированными боковыми карманами, а не частью воздушной камеры V-Core. Вентиляторы блоков питания забирают воздух через боковые окна, а штатный выхлоп HX1000 выходит через заднюю панель наружу. Поэтому БП не считаются потребителями давления КВД и не допускаются как путь утечки воздуха из нижнего отсека.

Оценка свободного объема КВД до учета вытеснения радиаторами, вентиляторами и помпами:

```text
V_lower_geometric = 500 x 300 x 260 = 39 000 000 мм3 = 0.0390 м3
V_psu_bays = 2 x 86 x 180 x 150 = 4 644 000 мм3 = 0.00464 м3
V_plenum_before_equipment_displacement = 0.0390 - 0.00464 = 0.03436 м3
```

Это изменение не требует увеличения габарита корпуса: БП занимают боковые зоны над радиаторным слоем и не выступают за внутренние `500 x 300 x 590 мм`.

## 5. Высота верхнего отсека

Верхний отсек ограничивается вертикальными мезонинами и трубной обвязкой.

```text
H_mezzanine = 236 мм
H_standoff = 40 мм
H_top_clearance = 54 мм

H_upper = 40 + 236 + 54 = 330 мм
```

Расчетно принимается 330 мм для верхнего отсека при ориентации мезонина 236 мм по вертикали и выводе труб без жесткого перегиба у крышки.

## 6. Итоговый размер

```text
Внутренний размер: 500 x 300 x 590 мм
Положение палубного пояса: 260 мм от нижнего края стоек
Вкладная палуба: 458 x 258 x 3 мм, область установки X = 21-479 мм, Y = 21-279 мм
Верхний отсек: 330 мм
Внешний размер рамы из профиля 20 x 20 мм: 540 x 340 x 590 мм
Внешний размер с ножками 40 мм: 540 x 340 x 630 мм
```

Итоговая спецификация для каркаса:

| Элемент | Длина, мм | Кол-во |
| --- | ---: | ---: |
| Вертикальная стойка | 590 | 4 |
| Горизонтальный профиль ширины | 500 | 6 |
| Горизонтальный профиль глубины | 300 | 6 |
| Стапель мезонина | 300 | 4 |
| Нижняя поперечина под БП/помпы | 500 | 2 |
| Опорный уголок палубы 20 x 20 x 2 | 458 | 2 |
| Опорный уголок палубы 20 x 20 x 2 | 258 | 2 |

Минимальная закупка профиля 20 x 20 мм остается 12000 мм: расчетная нарезка 9360 мм, остальное уходит на внутренние кронштейны, ошибку реза и технологический запас. Опорный уголок палубы закупается отдельно длиной 2000 мм.

## 7. Что нельзя резать без примерки

Точные отверстия под мезонины, водоблоки и сопла для обдува мезонинов нельзя переносить в DXF до примерки конкретных carrier board. Для платы-носителя в проекте нет официального механического даташита; размер 170 x 236 мм подходит для расчета корпуса, но не для координат крепежа.

До раскроя и сверления палубы нужно физически проверить:

1. Расположение SFF-8654 на обоих мезонинах.
2. Реальную высоту водоблоков V100 с фитингами.
3. Выводы питания carrier board.
4. Возможность уложить шланг 10/16 мм без перегиба у верхней крышки.

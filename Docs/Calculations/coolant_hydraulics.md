# Расчет гидравлических характеристик контура СЖО

Эффективный отвод 1.4 кВт тепловой мощности требует поддержания стабильного ламинарного или слаботурбулентного потока теплоносителя. Данный расчет обосновывает выбор компонентов и топологию соединений в системе NeuralTower.

## 1. Конфигурация контура и сопротивление компонентов

Система включает 7 активных гидравлических узлов. Принятые значения падения давления (Pressure Drop) при расходе 4 л/мин:

*   **Водоблок CPU (LGA 2011-3):** ~1.5 PSI.
*   **Водоблоки GPU V100 (4 шт.):** ~1.2 PSI на каждый.
*   **Радиаторы 360/240 мм (толщина 60 мм):** ~0.5 PSI на каждый.
*   **Фитинги и повороты (18 шт.):** суммарно ~0.8 PSI.

## 2. Топология соединений

Для минимизации суммарного сопротивления применена комбинированная схема:

1.  **Последовательный участок:** Помпа -> Водоблок CPU.
2.  **Параллельный участок:** Поток разделяется через Y-разветвитель на два плеча.
    *   Плечо А: Мезонин 1 (2 водоблока GPU последовательно).
    *   Плечо Б: Мезонин 2 (2 водоблока GPU последовательно).
3.  **Последовательный участок:** Объединение потока -> Радиатор 1 -> Радиатор 2 -> Резервуар.

### Расчет суммарного сопротивления (ΔP):
ΔP = ΔP_cpu + (ΔP_gpu * 2 / 2) + ΔP_rad1 + ΔP_rad2 + ΔP_fittings
ΔP = 1.5 + 1.2 + 0.5 + 0.5 + 0.8 = **4.5 PSI**.

## 3. Анализ производительности помпы D5

Типовая помпа стандарта D5 (например, Xylem/Laing) имеет следующие характеристики:
*   **Максимальный напор (Head pressure):** 3.9 - 4.5 метра водяного столба (~5.5 - 6.4 PSI).
*   **Максимальный расход:** 1500 л/ч (в свободном потоке).

При расчетном сопротивлении системы в **4.5 PSI**, помпа D5 будет работать в эффективной зоне своего графика, обеспечивая реальный расход теплоносителя на уровне **3.8–4.2 л/мин**. 

## 4. Теплофизические выводы

1.  **Скорость теплоносителя:** Расход ~4 л/мин при использовании шлангов с внутренним диаметром 10 мм обеспечивает скорость потока около 0.85 м/с. Этого достаточно для исключения застойных зон в микроканалах водоблоков V100.
2.  **Температурный градиент:** При тепловой нагрузке 1400 Вт и расходе 4 л/мин, разница температур воды между входом в первый водоблок и выходом из последнего радиатора составит:
    ΔT = Q / (m * Cp) = 1400 / (0.066 * 4180) ≈ **5.1°C**.
    Малый температурный дельт подтверждает высокую эффективность выбранной гидравлической схемы.

## 5. Рекомендации по эксплуатации

*   **Тип жидкости:** Использование специализированных диэлектрических антифризов с низкой вязкостью.
*   **Режим помпы:** Рекомендуется установка скорости помпы на уровень 4 (из 5) для баланса между ресурсом подшипника и производительностью.
*   **Контроль:** Включение в контур электронного датчика потока для аварийного отключения системы при падении расхода ниже 1.5 л/мин.

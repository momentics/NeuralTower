# Расчет длины SlimSAS SFF-8654 8i трассы

Расчет выполнен для корпуса 500 x 430 x 590 мм с палубой на высоте 260 мм. Геометрия корпуса взята из [case_dimensions.md](./case_dimensions.md).

## 1. Геометрия корпуса

| Параметр | Значение | Источник |
| --- | ---: | --- |
| Общая высота рамы | 590 мм | `frame_assembly.md` |
| Палуба | 260 мм от нижнего края стоек | `frame_assembly.md` |
| Материнская плата | 40 мм над палубой, Z = 300 мм | `deck_layout.md` |
| Мезонины | нижняя кромка примерно на Z = 300 мм | компоновка верхнего отсека |
| Стапели мезонинов | 300 мм между палубой и верхним поясом | `frame_assembly.md` |
| Габариты палубы | 500 x 430 мм | `deck_layout.md` |
| Окна для SlimSAS в палубе | 4 x (32 x 15 мм) | `deck_layout.md` |
| БП в КВД | на ребро, высота до 150 мм | `bom_list.md` |
| Радиаторы | Bykski CR-RD360RC-TK60-V2, 395.5 x 122 x 60 мм | `bom_list.md` |

## 2. Трасса одного кабеля

```text
Материнская плата (Z=300 мм)
    │
    │  ↓ 40 мм до палубы
    │
Палуба (Z=260 мм) ─── окно 32x15 мм
    │
    │  ↓ уходит в КВД
    │
КВД (Z=0-260 мм)
    │  └── обход БП, радиаторов и кабельных стяжек
    │
    │  ↑ поднимается обратно через палубу
    │
Мезонин (Z≈300 мм) ─── SFF-8654 у нижней кромки платы
```

Кабель не должен идти по кратчайшей прямой в верхнем отсеке. Его нужно опустить в КВД, провести по кабельной зоне и поднять к мезонину без резких перегибов у разъемов.

## 3. Вертикальный участок

| Участок | Длина |
| --- | ---: |
| Материнская плата -> палуба | ~40 мм |
| Палуба -> нижняя кабельная зона КВД | ~200 мм |
| Нижняя кабельная зона -> палуба | ~200 мм |
| Палуба -> коннектор мезонина | ~40 мм |
| Итого вертикаль | ~480 мм |

## 4. Горизонтальный участок

| Участок | Длина |
| --- | ---: |
| От адаптера PCIe/SlimSAS до зоны прохода в палубе | ~120-160 мм |
| Пролет внутри КВД с обходом БП и радиаторов | ~140-220 мм |
| От второго прохода в палубе до мезонина | ~40 мм |
| Итого горизонталь | ~300-420 мм |

## 5. Рабочая длина

```text
L_min = 480 + 300 = 780 мм
L_max = 480 + 420 = 900 мм
```

С учетом четырех радиусов изгиба, монтажного запаса и стяжек:

```text
L_work = 780-900 + 200 + 100 + 50 = 1130-1250 мм
```

Итог: кабели 1 м для новой рамы являются пограничными. Рабочая длина для заказа - 1.2 м. Если фактические разъемы на carrier board окажутся выше нижней кромки или кабель придется вести за блоками питания, лучше использовать 1.25-1.3 м.

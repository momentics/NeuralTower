## 2. Трасса одного кабеля SlimSAS

```
Материнская плата (Z=240 мм)
    │
    │  ↓ ~40 мм (до палубы)
    │
Палуба (Z=200 мм) ─── окно 32×15 мм
    │
    │  ↓ ныряет в КВД
    │
КВД (Z=0–200 мм)
    │  └── обход БП (высота ~150 мм на ребре)
    │  └── радиаторы (толщина 60 мм)
    │
    │  ↑ поднимается обратно через палубу
    │
Мезонин (Z=240 мм) ─── коннектор SFF-8654 у нижней кромки платы
```

**Важно:** Мезонины установлены на том же уровне, что и материнская плата — 40 мм над палубой. Разъёмы SFF-8654 8i расположены у нижней кромки мезонина, то есть на высоте Z ≈ 240 мм. Кабелю не нужно подниматься к верху мезонина — достаточно выйти из КВД и дойти до нижней кромки.

### 2.1. Вертикальный участок

| Участок | Длина |
|---------|-------|
| Материнская плата → палуба | ~40 мм |
| Палуба → дно КВД (обход БП) | ~150 мм |
| Дно КВД → палуба (подъём) | ~150 мм |
| Палуба → коннектор мезонина (нижняя кромка) | ~40 мм |
| **Итого вертикаль** | **~380 мм** |

### 2.2. Горизонтальный участок

Материнская плата расположена по центру. Мезонины — слева и справа. Адаптеры PCIe x16→2×SlimSAS вставляются в слоты PLX на плате.

| Параметр | Значение |
|----------|----------|
| Ширина внутреннего пространства | 350 мм |
| Расстояние от центра платы до мезонина | ~120–150 мм |
| Горизонтальный пролёт в КВД (обход БП) | ~100–150 мм |
| **Итого горизонталь** | **~220–300 мм** |

### 2.3. Суммарная длина трассы

```
L_мин = 380 + 220 = 600 мм
L_макс = 380 + 300 = 680 мм
```

С учётом:
- Радиуса изгиба SFF-8654 (минимум 25 мм на каждом повороте, ×4 поворота = +200 мм)
- Запаса на укладку и стяжки (+50–100 мм)
- Неидеальности монтажа (+50 мм)

```
L_рабочая = 600–680 + 200 + 100 + 50 = 950–1030 мм
```

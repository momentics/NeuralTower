# Расчет длины SlimSAS SFF-8654 8i

Расчет выполнен для корпуса 500 x 300 x 590 мм с палубой на высоте 260 мм. Геометрия корпуса взята из [case_dimensions.md](./case_dimensions.md).

## 1. Вывод

Шлейфы SlimSAS не прокладываются в вычислительной камере. Рабочая трасса должна уходить из верхнего отсека через герметизированные окна палубы, идти в подпалубном кабельном коридоре и возвращаться вверх к разъемам мезонинов.

Кабель не должен опускаться к дну КВД. Нижняя точка трассы находится сразу под палубой, на уровне примерно `Z = 225-235 мм`. Это сохраняет чистую зону вокруг плат и водоблоков, не перекрывает воздушные ножи и не заставляет кабель обходить радиаторы, блоки питания и помпы у основания корпуса. Рабочая трасса под палубой держится в центральной полосе `X = 90-410 мм` и перед боковыми PSU-отсеками, преимущественно в зоне `Y = 20-115 мм`.

Рабочий диапазон длины одного кабеля:

```text
L_work = 620-820 мм
```

Для закупки принимается стандартная длина 1000 мм, 8 шт. Длина 750 мм возможна только после макетной проверки конкретных адаптеров и ориентации разъемов.

## 2. Геометрия трассы

| Параметр | Значение |
| --- | ---: |
| Внутренняя ширина корпуса | 500 мм |
| Внутренняя глубина корпуса | 300 мм |
| Габарит ASUS X99-E WS | 305 x 267 мм |
| Палуба | Z = 260 мм |
| Плоскость материнской платы | около Z = 300 мм |
| Нижняя кромка мезонина | около Z = 300 мм |
| Рабочая плоскость подпалубного кабельного коридора | Z = 225-235 мм |
| Рабочая полоса кабельного коридора | X = 90-410 мм, Y = 20-115 мм |
| Спуск от разъема до палубы | около 40 мм |
| Спуск от палубы до кабельного коридора | 25-35 мм |

Разъемы SFF-8654 на адаптерах PCIe и на carrier board остаются в верхнем отсеке, но свободная длина шлейфа не идет по вычислительной камере. Выше палубы остается только короткий выход от разъема до ближайшего герметизированного прохода.

## 3. Маршрут кабеля

```text
PCIe/SlimSAS адаптер на ASUS X99-E WS
    |
    |  короткий выход от разъема, около 40 мм
    v
Герметизированное окно палубы, Z = 260 мм
    |
    |  спуск 25-35 мм
    v
Подпалубный кабельный коридор, Z = 225-235 мм
    |
    |  горизонтальная укладка под палубой
    v
Второе герметизированное окно палубы
    |
    |  подъем к разъему мезонина
    v
SXM2 carrier board, разъем SFF-8654 у нижней кромки
```

Подпалубный коридор должен проходить над нижним оборудованием. Он не использует дно КВД как кабельный лоток и не заводит шлейфы между радиаторами, БП и помпами. Боковые PSU-отсеки занимают `X = 0-86 мм` и `X = 414-500 мм`, поэтому сервисные петли SlimSAS фиксируются только в центральной зоне.

## 4. Расчет длины

### 4.1. Горизонтальный участок

Горизонтальная часть зависит от конкретного слота PCIe, стороны мезонина и положения герметизированных окон палубы. Для скрытой трассы нужно учитывать не кратчайшую прямую между разъемами, а путь между парой проходов через палубу.

| Сценарий | Длина |
| --- | ---: |
| Ближний адаптер -> ближайший мезонин через подпалубный коридор | 290-340 мм |
| Дальний слот или обход стоек/сопел под палубой | 350-390 мм |

Для расчета принимается:

```text
L_horizontal = 290-390 мм
```

### 4.2. Вертикальные участки

Кабель дважды проходит через палубу: вниз от адаптера и вверх к мезонину. При этом он не ныряет до отметки Z = 0 мм.

```text
L_vertical_adapter = 40 + 25-35 = 65-75 мм
L_vertical_mezzanine = 40 + 25-35 = 65-75 мм
L_vertical_total = 130-150 мм
```

### 4.3. Изгибы и монтажный запас

Для SFF-8654 нельзя закладывать резкий перегиб сразу после разъема и сразу после прохода через палубу. На четыре плавных перехода и фиксацию в подпалубном коридоре закладывается:

```text
L_bends = 120-160 мм
L_service = 80-120 мм
```

Сервисный запас хранится под палубой, закрепленным к кабельной планке или стяжечным площадкам. В вычислительной камере сервисная петля не допускается.

### 4.4. Итог

```text
L_min = 290 + 130 + 120 + 80 = 620 мм
L_max = 390 + 150 + 160 + 120 = 820 мм
```

Итоговая позиция для BOM:

```text
SlimSAS SFF-8654 8i: 1000 мм, 8 шт.
```

## 5. Требования к проходам через палубу

Для восьми шлейфов используются четыре герметизируемых окна 32 x 15 мм, по одному окну на пару кабелей. Окна размещаются рядом с зоной PCIe-адаптеров и рядом с нижними кромками мезонинов, чтобы верхний участок кабеля был минимальным.

Каждый проход закрывается разрезной резиновой или полимерной вставкой с прижимной планкой. После затяжки вставка должна обжимать оболочку кабеля и восстанавливать герметичность камеры давления. Радиусы кромок окон - не менее 2 мм, чтобы не повредить оболочку шлейфа.

## 6. Проверка при сборке

Перед закупкой нестандартной длины нужно проверить три вещи на макете:

1. Фактическую сторону выхода разъемов на PCIe/SlimSAS адаптерах.
2. Положение разъемов SFF-8654 на SXM2 carrier board.
3. Минимальный радиус изгиба конкретного кабеля при проходе через палубу.

Если все разъемы развернуты в сторону ближайших проходов, 750 мм может хватить без запаса. Для проектной спецификации принимается 1000 мм, потому что лишняя длина укладывается под палубой и не портит вычислительную камеру.

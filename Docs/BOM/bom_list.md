# Спецификация компонентов и материалов системы NeuralTower

Для реализации настольного вычислительного узла мощностью 1.4 кВт выбран перечень материалов, обеспечивающих жесткость конструкции и стабильный теплообмен. Все позиции подобраны с учетом интеграции профессионального серверного оборудования в кастомный алюминиевый каркас.

## 1. Силовая рама и конструкционные элементы

Основу скелета NeuralTower составляет алюминиевый профиль сечением 20 x 20 мм с пазом 6 мм. Размеры каркаса получены из компоновочного расчета, приведенного в [../Calculations/case_dimensions.md](../Calculations/case_dimensions.md). Для сборки требуются четыре вертикальные стойки по 590 мм, шесть профилей ширины по 500 мм и шесть профилей глубины по 300 мм. Внутренний объем корпуса составляет 500 x 300 x 590 мм, внешний размер рамы - 540 x 340 x 590 мм. Палуба устанавливается на высоте 260 мм от нижнего края стоек.

Внутри вычислительного отсека устанавливаются четыре вертикальных стапеля длиной по 300 мм, предназначенные для жесткой фиксации тяжелых мезонинов. В нижнем ярусе монтируются две дополнительные поперечины по 500 мм: нижняя на `Y = 115 мм`, `Z = 95 мм`, верхняя на `Y = 115 мм`, `Z = 245 мм`. Они формируют переднюю грань боковых PSU-отсеков, опоры помп и точки фиксации кабельных стяжек. Габаритные углы рамы собираются на 3-сторонних T-образных угловых кронштейнах 2020, остальные узлы - на двухсторонних угловых кронштейнах 2020. Дополнительная фиксация оборудования выполняется через Т-образные гайки М4/М5 и виброгасящие ножки высотой 40 мм, создающие необходимый зазор для забора воздуха.

Расчет крепежа рамы выполнен по схеме из [../../CAD/Frame/frame_assembly.md](../../CAD/Frame/frame_assembly.md). Нижний и верхний пояса образуют восемь внешних габаритных углов, поэтому для них требуется 8 трехсторонних кронштейнов. Палубный пояс находится на высоте 260 мм на уже установленных стойках и крепится двухсторонними кронштейнами: 4 угла x 2 примыкающих профиля = 8 шт. Внутренние стапели и нижние поперечины считаются отдельно: 4 стапеля x 4 кронштейна + 2 поперечины x 4 кронштейна = 24 шт. Итого двухсторонних кронштейнов: 8 + 24 = 32 шт.

## 2. Компоненты гидравлического контура

Система жидкостного охлаждения спроектирована под использование шланга из синтетического каучука EPDM с внутренним диаметром 10 мм. Циркуляцию обеспечивают два комбо-устройства Bykski B-UL-D5-X-V4, каждое из которых включает в себя помпу стандарта D5, резервуар 60 мм и защитный алюминиевый кожух. Теплосъем осуществляется парой медных радиаторов Bykski CR-RD360RC-TK60-V2 габаритом 395.5 x 122 x 60 мм, установленных горизонтально в основании корпуса длинной стороной вдоль ширины рамы.

Комплект теплосъемников включает водоблок для процессора LGA 2011-3 и четыре полнопоточных водоблока для ускорителей Tesla V100. Обдув радиаторов реализован шестью вентиляторами Thermaltake TOUGHFAN 12, нагнетающими воздух снаружи через нижнюю часть корпуса. Для сборки магистралей требуется 20 компрессионных фитингов стандарта 10/16 мм, 10 поворотных адаптеров на 90°, 4 адаптера на 45° и 3 Т-образных разветвителя. Герметичный переход шлангов через алюминиевую перегородку обеспечивают четыре проходных фитинга типа Fillport.

## 3. Интерфейсы данных и подсистема питания

Энергоснабжение системы разделено между двумя модульными блоками Corsair HX1000 Platinum, работающими в паре через синхронизатор Add2PSU. Блоки питания устанавливаются в изолированных боковых отсеках нижнего яруса: вентиляторная сторона получает воздух через боковые фильтры, а IEC-разъемы, персональные выключатели и выхлопные решетки выходят на заднюю панель. Передача данных от материнской платы ASUS X99-E WS к ускорителям выполняется восемью экранированными шлейфами SlimSAS SFF-8654 8i расчетной длиной 1000 мм. Основная длина шлейфов укладывается в подпалубном кабельном коридоре, поэтому кабели не проходят через вычислительную камеру. Для подключения кабелей используются четыре адаптера-разветвителя с PCIe x16 на два порта SlimSAS, поддерживающие режим разделения линий шины. Питание логики мезонинов реализовано через кастомные переходники с PCIe 8-pin на CPU 8-pin, изготовленные из медного провода сечением 0.82 мм².

## 4. Материалы корпуса и герметизации

Разделение температурных зон выполнено с помощью алюминиевой палубы толщиной 3 мм, в которой прорезаны прецизионные отверстия для воздушных ножей. Все пространство основания, не занятое радиаторами, перекрывается глухими алюминиевыми панелями для формирования герметичного пола. Герметичность стыков палубы и донных панелей с каркасом обеспечивается самоклеящимся резиновым уплотнителем на основе EPDM. Внешний облик и защиту узла формируют боковые акриловые панели толщиной 5 мм с магнитными нейлоновыми фильтрами, а также перфорированная верхняя крышка, обеспечивающая свободный выход нагретого воздуха из системы.

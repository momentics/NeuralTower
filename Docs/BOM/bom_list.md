# Спецификация компонентов и материалов системы NeuralTower

Для реализации настольного вычислительного узла мощностью 1.4 кВт выбран перечень материалов, обеспечивающих жесткость конструкции и стабильный теплообмен. Все позиции подобраны с учетом интеграции профессионального серверного оборудования в кастомный алюминиевый каркас.

## Силовая рама и конструкционные элементы

Основу скелета NeuralTower составляет алюминиевый профиль сечением 20х20 мм с пазом 6 мм. Для сборки каркаса требуются четыре вертикальные стойки по 550 мм, определяющие общую высоту устройства. Горизонтальная структура формируется шестью отрезками по 400 мм для обеспечения глубины и шестью отрезками по 350 мм, что задает итоговую ширину узла в 390 мм (350 мм внутренний размер + 2×20 мм профиля). Глубина внутреннего пространства составляет 380 мм (400 мм − 2×10 мм на прилегание палубы).

Внутри вычислительного отсека устанавливаются четыре вертикальных стапеля длиной по 300 мм, предназначенные для жесткой фиксации тяжелых мезонинов. В нижнем ярусе монтируются две дополнительные поперечины по 380 мм, на которые опираются блоки питания Corsair HX1000. Соединение всех элементов профиля выполняется торцевыми винтами М6х30 с предварительной нарезкой резьбы в центральном канале алюминиевых стоек. Дополнительная фиксация узлов обеспечивается литыми уголками, Т-образными гайками М4/М5 и виброгасящими ножками высотой 40 мм, создающими необходимый зазор для забора воздуха.

## Компоненты гидравлического контура

Система жидкостного охлаждения спроектирована под использование шланга из синтетического каучука EPDM с внутренним диаметром 10 мм. Циркуляцию обеспечивают два комбо-устройства Bykski B-UL-D5-X-V4, каждое из которых включает в себя помпу стандарта D5, интегрированный резервуар и защитный алюминиевый кожух. Теплосъем осуществляется парой одинаковых медных радиаторов типоразмера 360 мм и толщиной 60 мм, установленных горизонтально в основании корпуса.

Комплект теплосъемников включает водоблок для процессора LGA 2011-3 и четыре полнопоточных водоблока для ускорителей Tesla V100. Обдув радиаторов реализован шестью высоконапорными вентиляторами Noctua NF-A12x25 PWM, нагнетающими воздух снаружи через нижнюю часть корпуса. Для сборки магистралей требуется 20 компрессионных фитингов стандарта 10/16 мм, 10 поворотных адаптеров на 90°, 4 адаптера на 45° и 3 Т-образных разветвителя. Герметичный переход шлангов через алюминиевую перегородку обеспечивают четыре проходных фитинга типа Fillport.

## Интерфейсы данных и подсистема питания

Энергоснабжение системы разделено между двумя модульными блоками Corsair HX1000 Platinum, работающими в паре через синхронизатор Add2PSU. Передача данных от материнской платы ASUS X99-E WS к ускорителям выполняется восемью экранированными шлейфами SlimSAS SFF-8654 8i длиной 1 м. Для подключения кабелей используются четыре адаптера-разветвителя с PCIe x16 на два порта SlimSAS, поддерживающие режим разделения линий шины. Питание логики мезонинов реализовано через кастомные переходники с PCIe 8-pin на CPU 8-pin, изготовленные из медного провода сечением 18AWG.

## Материалы корпуса и герметизации

Разделение температурных зон выполнено с помощью алюминиевой палубы толщиной 3 мм, в которой прорезаны прецизионные отверстия для воздушных ножей. Все пространство основания, не занятое радиаторами, перекрывается глухими алюминиевыми панелями для формирования герметичного пола. Герметичность стыков палубы и донных панелей с каркасом обеспечивается самоклеящимся резиновым уплотнителем на основе EPDM. Внешний облик и защиту узла формируют боковые акриловые панели толщиной 5 мм с магнитными нейлоновыми фильтрами, а также перфорированная верхняя крышка, обеспечивающая свободный выход нагретого воздуха из системы.

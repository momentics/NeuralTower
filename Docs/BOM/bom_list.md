# Спецификация компонентов и материалов системы NeuralTower

Для реализации настольного вычислительного узла мощностью 1.4 кВт выбран перечень материалов, обеспечивающих жесткость конструкции и стабильный теплообмен. Все позиции подобраны с учетом интеграции профессионального серверного оборудования в кастомный алюминиевый каркас.

## 1. Силовая рама и конструкционные элементы

Основу скелета NeuralTower составляет алюминиевый профиль сечением 20х20 мм с пазом 6 мм. Размеры каркаса получены из компоновочного расчета, приведенного в [../Calculations/case_dimensions.md](../Calculations/case_dimensions.md). Для сборки требуются четыре вертикальные стойки по 590 мм, шесть профилей ширины по 500 мм и шесть профилей глубины по 430 мм. Внутренний объем корпуса составляет 500 x 430 x 590 мм, внешний размер рамы - 540 x 470 x 590 мм. Палуба устанавливается на высоте 260 мм от нижнего края стоек.

Внутри вычислительного отсека устанавливаются четыре вертикальных стапеля длиной по 300 мм, предназначенные для жесткой фиксации тяжелых мезонинов. В нижнем ярусе монтируются две дополнительные поперечины по 500 мм, на которые опираются блоки питания Corsair HX1000, помпы и кабельные стяжки. Основные узлы рамы собираются на внутренних угловых соединителях-сухарях для профиля 20x20 с пазом 6 мм. Дополнительная фиксация оборудования выполняется через Т-образные гайки М4/М5 и виброгасящие ножки высотой 40 мм, создающие необходимый зазор для забора воздуха.

## 2. Компоненты гидравлического контура

Система жидкостного охлаждения спроектирована под использование шланга из синтетического каучука EPDM с внутренним диаметром 10 мм. Циркуляцию обеспечивают два комбо-устройства Bykski B-UL-D5-X-V4, каждое из которых включает в себя помпу стандарта D5, резервуар 60 мм и защитный алюминиевый кожух. Теплосъем осуществляется парой медных радиаторов Bykski CR-RD360RC-TK60-V2 габаритом 395.5 x 122 x 60 мм, установленных горизонтально в основании корпуса.

Комплект теплосъемников включает водоблок для процессора LGA 2011-3 и четыре полнопоточных водоблока для ускорителей Tesla V100. Обдув радиаторов реализован шестью вентиляторами Thermaltake TOUGHFAN 12, нагнетающими воздух снаружи через нижнюю часть корпуса. Для сборки магистралей требуется 20 компрессионных фитингов стандарта 10/16 мм, 10 поворотных адаптеров на 90°, 4 адаптера на 45° и 3 Т-образных разветвителя. Герметичный переход шлангов через алюминиевую перегородку обеспечивают четыре проходных фитинга типа Fillport.

## 3. Интерфейсы данных и подсистема питания

Энергоснабжение системы разделено между двумя модульными блоками Corsair HX1000 Platinum, работающими в паре через синхронизатор Add2PSU. Передача данных от материнской платы ASUS X99-E WS к ускорителям выполняется восемью экранированными шлейфами SlimSAS SFF-8654 8i расчетной длиной 1.2 м. Для подключения кабелей используются четыре адаптера-разветвителя с PCIe x16 на два порта SlimSAS, поддерживающие режим разделения линий шины. Питание логики мезонинов реализовано через кастомные переходники с PCIe 8-pin на CPU 8-pin, изготовленные из медного провода сечением 18AWG.

## 4. Материалы корпуса и герметизации

Разделение температурных зон выполнено с помощью алюминиевой палубы толщиной 3 мм, в которой прорезаны прецизионные отверстия для воздушных ножей. Все пространство основания, не занятое радиаторами, перекрывается глухими алюминиевыми панелями для формирования герметичного пола. Герметичность стыков палубы и донных панелей с каркасом обеспечивается самоклеящимся резиновым уплотнителем на основе EPDM. Внешний облик и защиту узла формируют боковые акриловые панели толщиной 5 мм с магнитными нейлоновыми фильтрами, а также перфорированная верхняя крышка, обеспечивающая свободный выход нагретого воздуха из системы.

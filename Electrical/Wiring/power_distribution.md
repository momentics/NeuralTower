# Распределение питания между блоками HX1000

Надежная работа узла мощностью 1.4 кВт требует разделения нагрузки между двумя блоками питания Corsair HX1000. В системе используется схема с общим заземлением, где роль единой шины выравнивания потенциалов выполняет алюминиевый каркас из профиля 20 x 20 мм. Все металлические корпуса блоков питания должны иметь надежный электрический контакт с рамой.

Первый блок питания является ведущим. Он обеспечивает энергией материнскую плату, центральный процессор и первый мезонин. Второй блок функционирует как ведомый и питает второй мезонин, а также помпу системы жидкостного охлаждения. Синхронизация запуска осуществляется через адаптер Add2PSU, который замыкает контакты управления второго блока при появлении напряжения на периферийном разъеме первого.

Для питания логической части каждого мезонина необходимо задействовать по два разъема стандарта EPS (CPU 8-pin). Силовая часть графических процессоров подключается через стандартные разъемы PCIe 8-pin. Важным условием является использование отдельных кабельных линий от блоков питания для каждого входа мезонина во избежание перегрева контактов на стороне БП.

## Механическая ориентация БП и ввод 230 V

Оба Corsair HX1000 устанавливаются в изолированных боковых PSU-отсеках нижнего яруса, а не в центральном объеме КВД. Левый БП занимает `X = 0-86 мм`, `Y = 115-300 мм`, `Z = 95-245 мм`; правый БП занимает `X = 414-500 мм`, `Y = 115-300 мм`, `Z = 95-245 мм`.

Штатная задняя сторона каждого HX1000 с IEC C14, персональным выключателем и выхлопной решеткой должна быть обращена к задней панели корпуса. Сетевые кабели 230 V подключаются снаружи и не заводятся внутрь корпуса. Модульная сторона БП смотрит внутрь, к центрально-передней сервисной зоне, откуда DC-жгуты сразу фиксируются на кабельной планке под палубой.

Вентилятор каждого БП работает по собственному наружному контуру: боковой фильтр -> HX1000 -> задний выхлоп. Этот контур не должен сообщаться с КВД и не используется для охлаждения материнской платы, мезонинов или радиаторов СЖО.

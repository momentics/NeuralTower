# Маркировка SlimSAS и физическая топология GPU

Здесь зафиксировано соответствие между слотами ASUS X99-E WS, PLX-коммутаторами, адаптерами PCIe x16 -> 2x SlimSAS, мезонинами и порядком GPU. Этот порядок используется при сборке, проверке `nvidia-smi topo -m` и настройке `CUDA_VISIBLE_DEVICES`.

Каждый слот PCIe x16 материнской платы через пассивный адаптер разветвляется на два кабеля SlimSAS SFF-8654 8i, которые идут к одному GPU на мезонине. Перепутанные кабели приведут к перекрестному соединению линий PCIe и невозможности инициализации видеокарт.

Сама оплетка кабелей обычно черная. Маркировка наносится цветными термоусадочными кольцами или изолентой рядом с каждым разъемом SFF-8654. Цвет привязан к GPU, а не к кабелю: оба разъема, ведущие к одному GPU, получают одинаковый цвет. Внутри пары порт A и порт B различаются белой наклейкой с буквой.

## 1. Цветовая карта

```text
GPU 0 - красный
GPU 1 - синий
GPU 2 - зеленый
GPU 3 - желтый
```

## 2. Таблица соответствия

| GPU | Цвет кольца | Слот материнской платы | Чип PLX | Адаптер | Мезонин | Порт A -> | Порт B -> |
| --- | --- | --- | --- | --- | --- | --- | --- |
| GPU 0 | Красный | PCIEX16_1 | PLX1 | Адаптер 1 | Мезонин 1 | GPU 0, порт A | GPU 0, порт B |
| GPU 1 | Синий | PCIEX16_3 | PLX1 | Адаптер 2 | Мезонин 1 | GPU 1, порт A | GPU 1, порт B |
| GPU 2 | Зеленый | PCIEX16_5 | PLX2 | Адаптер 3 | Мезонин 2 | GPU 2, порт A | GPU 2, порт B |
| GPU 3 | Желтый | PCIEX16_7 | PLX2 | Адаптер 4 | Мезонин 2 | GPU 3, порт A | GPU 3, порт B |

## 3. Схема

```text
Материнская плата ASUS X99-E WS

PLX1:
  PCIEX16_1 -> адаптер 1 -> красная пара SlimSAS -> GPU 0, порт A/B
  PCIEX16_3 -> адаптер 2 -> синяя пара SlimSAS  -> GPU 1, порт A/B

PLX2:
  PCIEX16_5 -> адаптер 3 -> зеленая пара SlimSAS -> GPU 2, порт A/B
  PCIEX16_7 -> адаптер 4 -> желтая пара SlimSAS  -> GPU 3, порт A/B

Мезонин 1:
  GPU 0 и GPU 1 связаны NVLink и формируют первую локальную пару.

Мезонин 2:
  GPU 2 и GPU 3 связаны NVLink и формируют вторую локальную пару.
```

## 4. Правило физической укладки

Маркировка кабелей не меняет правило трассировки: шлейфы SlimSAS не ведутся через вычислительную камеру. От разъема кабель сразу уходит через ближайший герметизированный проход палубы, идет в подпалубном кабельном коридоре на уровне `Z = 225-235 мм` и возвращается вверх у соответствующего мезонина. Трасса не опускается к дну КВД и не проходит через зону БП, радиаторов и помп. Боковые PSU-отсеки занимают `X = 0-86 мм` и `X = 414-500 мм`, поэтому сервисные петли SlimSAS закрепляются в центральной полосе `X = 90-410 мм`.

## 5. Правила сборки

1. Цвет определяет GPU. Оба конца каждого кабеля помечаются кольцом одного цвета, соответствующего целевому GPU.
2. Буква определяет порт. Наклейка `A` или `B` гарантирует, что порт A адаптера идет в порт A GPU, а порт B - в порт B.
3. Перед включением проверьте, что на каждом GPU оба входящих разъема имеют одинаковый цвет кольца.
4. Перед подачей питания прозвоните каждый кабель от слота материнской платы до правильного GPU.
5. `CUDA_VISIBLE_DEVICES` должен отражать физический порядок `"0,1,2,3"`, где 0 и 1 находятся на мезонине 1, а 2 и 3 - на мезонине 2. Этот порядок критичен для корректной работы TP=2 + PP=2 в vLLM.
